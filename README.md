Project Overview:

A Python-based tool that evaluates the strength of a password based on predefined criteria.

Features:

1. Checks password length

2. Validates presence of uppercase, lowercase, numbers, and special characters

3. Provides strength feedback (Weak / Moderate / Strong)

Concepts Used:

1. String handling

2. Conditional logic

3. Loops

4. Input validation

How to Run:

1. Run the script.

2. Enter a password.

3. The program evaluates and displays its strength.

Learning Outcome

This project enhanced understanding of string manipulation and logic-based validation techniques.
